# Vocanetic – VB-Based Quiz Application

## 📌 Description
Vocanetic is a quiz-based application developed using Visual Basic as part of a Visual Programming course. This application is designed to support interactive learning, especially for students in the Software Engineering (RPL) major.

## 🎯 Purpose
The purpose of this application is to:
- Provide an interactive quiz platform
- Improve students' understanding through practice questions
- Support engaging and user-friendly learning experiences

## ⚙️ Features
- Multiple-choice quiz questions
- Score calculation
- Simple and user-friendly interface
- Interactive navigation between questions

## 🛠️ Built With
- Visual Basic
- Microsoft Visual Studio

## 📚 Project Type
Academic Project (Visual Programming Course)
